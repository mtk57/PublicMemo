ブール代数


1. 基本的な演算子:

ブール代数では、以下の3つの基本的な演算子が使用されます。

●AND (論理積): 
AとBの両方が真(1)の場合のみ、結果が真(1)になります。
記号は「・」または「∧」または単にABと表記します。


●OR (論理和): 
AとBの少なくとも一方が真(1)の場合、結果が真(1)になります。
記号は「＋」または「∨」と表記します。


●NOT (否定): 
Aが真(1)の場合は偽(0)に、Aが偽(0)の場合は真(1)にします。
記号は「¬」または「¯」または「!」と表記します。


2. 基本的な法則:

以下に、ブール代数の基本的な法則を示します。

●交換法則:

A + B = B + A
A ・ B = B ・ A (AとBの順番を入れ替えても結果は同じ)


●結合法則:

(A + B) + C = A + (B + C)
(A ・ B) ・ C = A ・ (B ・ C) (複数の演算を行う際に、どの順番で計算しても結果は同じ)


●分配法則:

A ・ (B + C) = (A ・ B) + (A ・ C)
A + (B ・ C) = (A + B) ・ (A + C) (掛け算と足し算の関係を表す)


●恒等法則:

A + 0 = A
A ・ 1 = A (0は足し算の、1は掛け算の単位元)


●補元法則:

A + ¬A = 1
A ・ ¬A = 0 (ある値とその否定のORは常に真、ANDは常に偽)


●べき等法則:

A + A = A
A ・ A = A (同じ値同士のORやANDは元の値と同じ)


●吸収法則:

A + (A ・ B) = A
A ・ (A + B) = A (複雑な式を簡略化する際に役立つ)


●ド・モルガンの法則:

¬(A + B) = ¬A ・ ¬B
¬(A ・ B) = ¬A + ¬B (ORの否定は各々の否定のAND、ANDの否定は各々の否定のOR)
